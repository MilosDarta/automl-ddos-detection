# Обнаружение DDoS-атак с помощью AutoML на наборе данных CIC‑DDoS2019

Сравнение методов многоклассовой классификации сетевого трафика для выявления DDoS‑атак:

- **Baseline**: LightGBM с весами классов (ручной бейзлайн)
- **AutoGluon** (Amazon)
- **LightAutoML** (Sber AI Lab)

>  **Примечание:** LightAutoML может требовать специфических версий зависимостей. Рекомендуется использовать **Python 3.10** и `lightautoml>=0.6.0`.  
> Эксперименты проводились в среде Google Colab (CPU, 12.7 ГБ RAM) с временными лимитами 60 минут.

## 📊 Набор данных

Использован [CIC‑DDoS2019](https://www.unb.ca/cic/datasets/ddos-2019.html) – 431 371 сетевой поток, 78 исходных признаков. После предобработки (удаление идентификаторов, низковариативных и коррелированных признаков, объединение редких атак) остаётся 55 признаков и 10 классов:

| Класс         | Количество | Доля, % |
|---------------|------------|---------|
| DrDoS_NTP     | 121 368    | 28,14   |
| TFTP          | 98 917     | 22,93   |
| Benign        | 97 831     | 22,68   |
| Syn           | 49 373     | 11,45   |
| UDP           | 18 090     | 4,19    |
| Rare_Attack   | 11 765     | 2,73    |
| DrDoS_UDP     | 10 420     | 2,42    |
| UDP‑lag       | 8 872      | 2,06    |
| MSSQL         | 8 523      | 1,98    |
| DrDoS_MSSQL   | 6 212      | 1,44    |

## 📈 Основные результаты (30 мин)

| Модель               | Время (с) | F1‑macro |
|----------------------|-----------|----------|
| LightGBM Baseline    | 105       | **0.803**|
| AutoGluon            | 1801      | 0.797    |
| LightAutoML          | 1176      | 0.755    |

Ручной LightGBM с весами классов показал наилучшее соотношение качество/время.
